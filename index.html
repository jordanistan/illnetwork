ill.network Security Scanner

A professional-grade, comprehensive security and health audit tool for Linux systems. Run it from anywhere, scan anything.

Start Scan

Run a Scan

To initiate a scan, run the following command in your terminal. The results will be displayed here and saved to a report file.

docker compose exec scanner ./healthcheck.sh -t <target_host>

Replace <target_host> with the IP address or hostname of the machine you want to scan. To scan the local machine (inside the container), use localhost.
