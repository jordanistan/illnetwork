Get Started in Seconds

Run the following command in your terminal to download, set up, and run the scanner automatically. It will clone the repository and start the services for you.

curl -sSL https://raw.githubusercontent.com/jordanistan/illnetwork/main/setup.sh | bash

Once the setup is complete, you can run a scan using the command provided in the terminal.
